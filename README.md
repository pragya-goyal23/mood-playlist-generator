Mood-Based Playlist Generator

A simple menu driven Python program that creates playlists based on the user’s mood.
It allows viewing moods, adding moods, adding songs, and generating a shuffled playlist.
The project includes a small loading animation to make the experience smoother.

⸻

-> Overview

This project is part of the Introduction to Programming & Problem Solving course.
It demonstrates basic Python problem-solving skills using:
	•	Dictionaries
	•	Loops
	•	Conditional statements
	•	Functions
	•	Randomization
	•	Simple animations using time.sleep()

The program runs in the terminal and helps users manage mood-wise playlists.

⸻

-> Features
	•	View all available moods
	•	Add new moods
	•	Add songs to existing moods
	•	Generate a playlist for any mood
	•	Shuffle playlist songs
	•	Simple “Processing…” loading animation
	•	Clean and beginner-friendly code structure

⸻

-> Technologies / Tools Used
	•	Python 3
	•	time module (for loading animation)
	•	random module (for shuffling songs)
	•	Terminal / Command Line

⸻

-> How to Run the Project
	1.	Install Python 3 (if not already installed)
	2.	Download the project folder
	3.	Open a terminal in the project location
	4.	Run:
  The menu will appear, and you can interact with it using numeric choices.


-> Instructions for Testing

Test the program by trying the following:
	•	View moods
	•	Add a new mood (example: “focus”)
	•	Add 2–3 songs to that mood
	•	Generate a playlist
	•	Check if songs shuffle each time
	•	Try invalid moods to confirm error messages work
